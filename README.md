## 1. Découpage de namespace
```
.
├── namespace_dev
│   ├── app
│   │   ├── configMap.yml
│   │   ├── deployment-app.yml
│   │   ├── ingress-app.yml
│   │   └── service-app.yml
│   ├── quota-mem-cpu.yml
│   └── redis
│       ├── deployment-redis.yml
│       └── service-redis.yml
└── namespace_hello
    └── hello_world
        ├── deployment-hello_world.yml
        ├── ingress-hello_world.yml
        ├── quota-mem-cpu.yml
        └── service-hello_world.yml

```
(en gros) ça ressemble à ça:
```bash=
$ kubectl get namespace

kube-system       Active   5h20m
```
